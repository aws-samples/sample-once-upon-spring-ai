# Theme Texture Assets

Place texture images here for the D&D tabletop theme:

- `parchment-bg.png` — aged parchment paper, used for panels and form backgrounds
- `leather-bg.png` — dark brown leather, used for UI panel backgrounds
- `wood-bg.png` — dark oak wood plank, used for the app shell background

See the design document for image generation prompts.

If textures are missing, the theme falls back to solid CSS colors (e.g. `--color-parchment`, `--color-wood-dark`).
